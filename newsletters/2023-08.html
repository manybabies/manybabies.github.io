MB3: Rule Learning

- Its not too late to join our data collection team! The current testing deadline is Dec. 31, but we have a mechanism for extending this deadline if needed. Email Ingmar to sign up or for more info.

- Were also looking for folks to contribute to the MB3 analysis team - e.g. with scripts for data validation, data merging, code documentation, code reviewing, plotting data, scrambling data, etc. Please email Ingmar if youre interested!

- The MB3 analysis team is still looking for pilot data to use in fine-tuning the analysis pipeline. Please submit using the data upload form. Any data is helpful!

- The MB3 spin-off project MB3T (Test-retest) is collecting data! There is still time to join! Please check the MB3T Lab Manual and email Anna Exner to indicate your interest.

MB5: Hunter & Ames model of Infant Attention

We recently received feedback from the MB5 community on the revised manuscript to Nature Human Behaviour and are currently working to incorporate these edits. We will be resubmitting to the journal at the end of August.
